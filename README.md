# xau-live-mirror

Public mirror of the latest `OANDA:XAUUSD` TradingView webhook price.

## Safe access

- Human/browser view: https://github.com/lbronight/xau-live-mirror/blob/main/latest.json
- GitHub Contents API: https://api.github.com/repos/lbronight/xau-live-mirror/contents/latest.json?ref=main
- ChatGPT/Codex: read `latest.json` through the connected GitHub app from repository `lbronight/xau-live-mirror`.

Do not open the `raw.githubusercontent.com` URL on this computer. Avast currently classifies that domain as `URL:Phishing`; the mirror itself remains healthy and continues updating through the GitHub Contents API.
